Voting

In between games, you will be given the opportunity to vote. The map/mode with the most votes wins. If there are not enough votes, the server will choose the next map in its map rotation list.

You can cast a vote at any time by pressing F2. If enough other players vote for the same map and mode, the new map will be loaded immediately.

Master status

If no-one has taken master status yet, you may take it by typing /setmaster 1 . This will give you the power to /kick problematic users and change the /mastermode. Please be a fair master, as it spoils the game for everyone else if a master abuses their power. The server's administrator will take action if someone is found to be abusing master status.

Controlling the game

Once you have master status, you have access to additional commands to help administer the server.

About Client Numbers (cn's)

Every player on the server has a Client Number (cn). You can pass this cn to commands to perform actions on a player, for example /spectator or /kick

Mastermode

You may change the mastermode as follows:

  • /mastermode 0: normal mode, like other players.
  • /mastermode 1: veto mode. You can now change maps at will by voting.
  • /mastermode 2: locked. As with 1 but new players will join as spectators and will be unable to play until you unspectate them
  • /mastermode 3: private. As with 1, but new players cannot join this server.

Spectator

Players can enter spectator mode by typing /spectator 1. In addition, you can put other players in spectator mode by typing /spectator 1 <cn><tt>. To take a player out of spectator mode type <tt>/spectator 0 <cn>

Kick

You can kick (short-term ban) a player by typing /kick <cn>. The ban will last until the server is empty or until you type /clearbans. As a rule of thumb, you should only kick a player if they are ruining the fun for other players (eg obvious cheating, offensive behaviour). Please do not kick players because they are better or you don't like them. Remember: what goes around comes around.

Bots

Each map has a "normal" player capacity, and by default the server will use bots to make up that capacity if there are not enough players. You can adjust this using /botbalance. For example, on an 8 player map with 2 human players, /botbalance 1 will add 6 bots. /botbalance 0.5 will add two bots. /botbalance 0 will remove all bots from the server.

You can also adjust the skill level of the bots with /botminskill (default 80) and /botmaxskill (default: 100).

Other fun commands

Kamikaze

Kamikaze mode affects whether players blow up when killed. It can be any of the following (default is 1):

  • /kamikaze 0: Never blow up.
  • /kamikaze 1: Player blows up when killed if they are holding a grenade at the time.
  • /kamikaze 2: Player blows up when killed if they possess any grenades, regardless of whether it is being held at the time.
  • /kamikaze 3: Player always blows up when killed.
