Texture Blending in Maps

Two textures can be smoothly blended together to create variations on architecture and terrain.


Defining Blended Textures:

If you are using the default package.cfg for editing maps in Blood Frontier, several textures and combinations will already be set up for you. Texture thumbnails that have an additional texture layer to blend will have the texture they are set to blend with in one of the corners in the texture browser (f2).


If you are not using the default package, you will need to set your own textures up. Texture blending can use any two textures that are already set in your map.cfg, but you need to define the textures normally first.


Next, in order to set up textures to blend, in your map cfg, under the texture that you want to have a second texture layer, you would add


texlayer N


where N is the slot taken by the second texture you want to add. Keep in mind that in Blood Frontier the first *7* slots are taken by default textures such as sky, water, and the six basic architectural textures.


So, if you have defined 50 new textures, and the 51st texture is something that you want to paint with, then under any texture you want to blend it with, you would enter


texlayer 58


Brushes

There will be a number of brushes already included and set up in the default Blood Frontier package.cfg. If you want to set up your own brush, simple create a gray scale PNG file to use as a pattern. Keep in mind that the size of the brushes are relatively large, and not meant for super detailed work. So a 16 x 16 size image will make a brush that covers a very large patch of architecture.


Adding a New Brush

In your map.cfg, add


addblendbrush N F


Where N is the name of your brush, that you will use later to load the brush, and F is the file name, relative to the data directory. So if you have a directory in data called "mydir", you have put yuor brushes into a sub-directory called "brushes", and the brush you want to add is called "crack.png", then you would add the following line to your map.cfg file:


addblendbrush crack "mydir/brushes/crack.png"


Loading a Brush

Enter editing mode by pressing f1. If you see hands in front of you, press f1 again. Open the console with the tilde key and type


/setblendbrush N
remember that the slash automatically appears by default when you enter the console.


Where N is the name of your brush that you defined in the cfg, or the name of an existing brush. So to set the brush to the one that was defined above, you would open the console with the tilde key and type


/setblendbrush crack
remember that the slash automatically appears by default when you enter the console.


Start Painting Already!

First, select a texture that you have defined a secondary texture for.


Second, you will need to be in editing mode, on a NEW map. Keep in mind that the texture blending is not set up in older map.cfg files, because the capability did not exist yet.


Third, texture the surfaces that you will want to paint with the textures that have had an extra texture layer defined for them in your map.cfg. You can only paint on textures that have had a second layer defined in the cfg.


Now, you need to turn on painting mode. Press tilde to enter the console and type


/blendpaintmode N
remember that the slash automatically appears by default when you enter the console.


Where N is a number from 0 to 5, which defines how the painting will be done. Setting it to 0 turns paint mode off.

  • 0 - off
  • 1 - set
  • 2 - min(destination, source)
  • 3 - max(destination, source)
  • 4 - destination - source
  • 5 - destination + source


Experiment with the numbers to understand what they do.


Finally, in order to paint on to the surface, you would open the console and type


/paintblendmap
remember that the slash automatically appears by default when you enter the console.


If you've done everything right, you should see your first blended textures. If not, check out the IRC forum. Most of the community members there are willing to offer help.


Oops - I Screwed Up!

There is NO UNDO for texture blending. If you've screwed something up, you can do one of two things. Open the console with the tilde key and type


/clearblendmap
remember that the slash automatically appears by default when you enter the console.


This will delete ALL of the texture blending for the ENTIRE level.


/clearblendmapsel
remember that the slash automatically appears by default when you enter the console.


This will clear the texture blending on the selected geometry.


Hints and Tips

  • The amount of rendering passes the engine is required to make doubles on areas where the textures have been blended.
  • DON'T start working with blended textures until you are happy with the architecture in the level. They are still new and may cause unforeseen Issues.
  • SAVE OFTEN. You are working with a new feature, after all.
