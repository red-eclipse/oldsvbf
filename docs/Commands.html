Input and Camera Control

Before changing the controls, one first has to realise, there's several camera states, as listed below.

Camera State Description
Editmode The editmode camera state is entered by entering editmode.
First Person the first person state is used when the player is not in third person, nor doing any of the other stuff.
Prone Prone is entered when a player attempts to zoom while not in posession of a rifle.
Snipe Snipe is entered when a player attempts to zoom while posessing a rifle.
Spectator The spectator state is used when a player is spectating.
Third Person The third person camera state is used when a player is not in first person mode

Editmode

  • Template:Intvar
Sets the mousetype for editmode, 0 is standard FPS mode, 1 is when the screen pans, to the relative cursor (small dot), and 2 is when the crosshair is fixed to the mouse, and the screen pans, according to that.
  • Template:Intvar
Sets the FOV, a high FOV means more can be seen at once, a lower FOV means less can be seen at once.
  • Template:Intvar
Determines how far from the centre of the screen you have to move the mouse before the screen pans, when you're using editmouse 1 or 2.
  • Template:Intvar
Determines at what speed the camera will rotate when editmouse is 1 or 2.

Sets how much damping is applied on the swaying speed of the weapons, when you move about.

gamemodes and mutators

Below is a table of gamemodes and mutators. Mutators is simply a value that can define special bitwise game properties, gamemodes define which mutators are available, and what other game features are active.

eg, giving a game a mutator value of 6 means team, instagib and team will be active.
Gamemodes and mutators are set by typing Template:Cmd, and then followed by a Template:Cmd. G stands for the gamemode ID, M the total if the mutators you want, added together, and N being a map name. Like for the above example Template:Cmd;Template:Cmd.

Bit wise value Description
1 Multi sided - 4 teams partake in the match
2 Team - 2 teams partake in the match
4 Instagib - 1 hit from any weapon is equivalent to a kill/frag
8 Duel - 2 players spawn, a challenger and the champion (the winner of the round is the champion)
16 Last Man Standing - everyone has one life, use it wisely

</table>

Below is the table for gamemode information. with the mutator boxes, N means NOT available, C means compulsory, or forced, O means optional.

Gamemode Mode name Description Multi-Sided Team Instagib Duel Last Man Standing
0 Demo Plays back recorded demos N N N N N
1 Lobby The default mode, simply, a mode to say hi and explore maps with N N N N N
2 Editmode Allows you to edit maps, whether this be by yourself, or cooperatively with others N N N N N
3 Mission Play a Mission Cooperatively or alone. Not implemented yet! N N N N N
4 Deathmatch You fight, and you shoot that guy's brains out, simple. Y Y Y Y Y
5 Secure the Flag (StF) The flags act as bases and take several seconds to claim ownership of.
You then gain points. Whoever claims ALL the flags or has the most points at the end of a match, wins.		 Y C Y N N
6 Capture the Flag (CtF) Each team has a flag, obviously the idea is to capture the flag of the enemy teams as many times as possible. Y C Y N N

IRC

Bloodfrontier has built in IRC, along with a built in IRC bot that relays server information to an IRC channel, the verbosity can be set to either 1 or 2, note that 0 is off, and is set per channel. to use the bfbot, one must connect to the server using addircrelay, and join an IRC channel, while providing a relay value above 0.

note that Name refers to a name you assign to it, eg freenode, while you made the server irc.freenode.net

  • Template:Cmd
joins an IRC network, under the identity of the provided alias
  • Template:Cmd
joins an IRC network, under the identity of the provided alias, but also relays all received messages to the console, and relays most console messages to connected IRC channels
  • Template:Cmd
  • Template:Cmd
joins an IRC channel, and tries to join the channel every second when it's not connected to the channel
  • Template:Cmd
joins an IRC channel, but does not persist on staying connected
  • Template:Cmd
the IRC gui, an argument can be provided as a filter to only show the matching IRC network
