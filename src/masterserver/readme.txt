CubeEngine compatible masterserver lister.
Made for BloodFrontier.
Developed by Drakas Tralas <drakas.tralas@googlemail.com> / Vykintas Narmontas <vykintas.narmontas@googlemail.com>
All code is released under the MIT license.

Requirements:
* PHP 5
* Sqlite3 through PHP-PDO
* .htaccess support under Apache (or any alike rewriting support)

Proper caching has not been implemented as of yet (and might not be).
